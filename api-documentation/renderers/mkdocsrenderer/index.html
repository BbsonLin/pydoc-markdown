MkdocsRenderer

class MkdocsRenderer(Struct)

output_directory

The output directory for the generated Markdown files. Defaults to build/docs.

clean_docs_directory_on_render

Remove the docs directory in the output_directory before rendering.

pages

The pages to render into the output directory.

markdown

Markdown renderer settings.

site_name

The name of the site. This will be carried into the site_name key of the mkdocs_config.

mkdocs_config

Arbitrary configuration values that will be rendered to an mkdocs.yml file.

organize_modules

 | organize_modules(graph) -> Iterable[Tuple[List[str], Page, List[Object]]]

Organizes the objects in the graph in pairs with the pages that are supposed to contain them (per the "contents" filter).
